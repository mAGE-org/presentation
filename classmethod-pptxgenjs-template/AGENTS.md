# スライド作成ガイド

このプロジェクトではクラスメソッドのブランドテーマに沿った PPTX スライドを作成します。

## テンプレート

`template/` ディレクトリにクラスメソッドテーマのテンプレートがあります。

- `template/theme.js` — カラー・フォント・レイアウトの定数定義
- `template/generate-template.js` — テンプレート .pptx を生成するスクリプト
- `template/TEMPLATE.md` — 全18レイアウトの詳細仕様

## 前提条件

Anthropic 公式の pptx スキルがインストール済みであること。

https://github.com/anthropics/skills/tree/main/skills/pptx

## スライドの作成・編集

pptx スキルの SKILL.md を読み、指示に従ってください。

### 新規作成: PptxGenJS でゼロから作成

pptxgenjs.md の手順に従い、`template/theme.js` の定数を利用してスクリプトで直接作成します。

```javascript
const { COLORS, FONTS, SIZE, SLIDE, LAYOUT } = require("./template/theme");
```

### 既存スライドの修正

PptxGenJS で作成したスライドの場合は、生成スクリプト（.js）を編集して再実行してください。

PptxGenJS 以外で作成された .pptx ファイルを修正する場合は、editing.md の手順に従い、XML 編集ワークフローを使用します。

1. 既存の .pptx を展開（unpack）
2. 各スライドの XML を編集
3. クリーン＆パック（pack）して .pptx を出力

### QA を実施する

新規作成・既存修正いずれの場合も、pptx スキルの QA セクションに従い、必ずビジュアル検査を行ってください。

## レイアウト選択の指針

スライドの内容に合わせてレイアウトを選んでください。同じレイアウトの繰り返しは避けること。

| 内容 | 推奨レイアウト |
|------|-------------|
| 表紙 | 1: タイトルスライド |
| 章の切り替え | 2: セクション区切り / 17: 青アクセント |
| 箇条書き・説明 | 3: 標準コンテンツ / 4: ヘッダーなし |
| 図表・スクリーンショット | 5: 画像中央 / 6: テキスト＋図（右） / 7: テキスト＋図（左） |
| 対比・並列 | 8: 2カラム / 9: 3カラム / 12: 比較 |
| 数値・KPI | 10: 主要指標 |
| 手順・フロー | 11: プロセス・手順 |
| 機能・特徴一覧 | 13: アイコングリッド |
| 引用・強調メッセージ | 14: 引用 |
| 目次 | 15: アジェンダ |
| データ表 | 16: テーブル |
| 締め | 18: クロージング |

## デザインルール

- テキストカラーは `GRAY_MEDIUM`（`434343`）を基本とする
- アクセントカラーは `BLUE`（`2C67E5`）を使う
- 背景はホワイト（`FFFFFF`）またはアクセントグレー（`F3F3F3`）
- フォントは `Noto Sans JP` で統一
- スライドサイズは 16:9（`LAYOUT_16x9`）
- ロゴは `generate-template.js` 内に埋め込み済み（クラスメソッド公式SVG）
